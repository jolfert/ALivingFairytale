# A Living Fairytale Site Review Guide

This document is for the site owner to review the current website build and mark up any changes.

The goal is simple:
- confirm what feels right
- flag anything that feels off
- note any missing characters, photos, wording, or booking details

You do not need to know code to use this document.

## How To Review

Please review the site page by page and add notes directly under each section.

Helpful note format:

```text
Change:
Why:
Priority: High / Medium / Low
```

Example:

```text
Change: Replace Belle's main photo with the newer gold-dress image.
Why: The current one feels darker than the rest of the page.
Priority: High
```

## What We Are Reviewing

Please keep an eye on:
- overall first impression
- whether the site feels easy to navigate
- whether the page feels too busy or too wordy
- whether each character uses the correct photos
- whether each character name is correct
- whether any active characters are still missing
- whether pricing / package language feels right
- whether booking feels simple and trustworthy
- how everything feels on mobile
